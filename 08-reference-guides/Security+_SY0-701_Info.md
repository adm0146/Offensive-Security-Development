## üéØ SY0-701 Exam Tips

### **Performance-Based Questions (PBQs):**
- 3-5 PBQs on the exam
- Worth more points than multiple choice
- Can take 5-10 minutes each
- **Strategy:** Skip them initially, do multiple choice first, then return

**Common PBQ Types:**
- Network diagram analysis
- Firewall rule configuration
- Log file analysis
- Certificate selection
- Security control implementation

### **Multiple Choice Strategy:**
- Read carefully for "BEST" vs "MOST secure"
- Eliminate obviously wrong answers
- Watch for double negatives
- Flag uncertain questions for review
- Manage your time (1 min per question)

### **Memorization Requirements:**
- **Ports:** 20+ common ports (HTTP=80, HTTPS=443, SSH=22, etc.)
- **Protocols:** Know which are secure vs insecure
- **Acronyms:** 100+ acronyms (CIA, AAA, PKI, SIEM, etc.)
- **Encryption:** Algorithm names, key sizes, use cases
- **Attack types:** Recognize descriptions of attacks
